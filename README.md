# Handprint

Handprint is a local civic identity engine. It helps people discover useful action within reach, participate with verified organizers, and grow a visible personal handprint from real-world contribution.

The working thesis:

> You cannot solve all the world's problems. But you can impact the world within your reach.

## Product Wedge

Start narrow:

- One launch community
- Verified local action discovery
- RSVP and check-in
- Organizer event creation
- A beautiful handprint visualization that grows with participation

Avoid starting as a general social network, national activism platform, or broad event marketplace.

## Local Development

