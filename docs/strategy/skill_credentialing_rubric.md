# Skill Credentialing Rubric

Handprint skills should not all behave like self-declared profile tags. Some are safe entry points for new users, some should be earned through reliable participation, and some require training, documents, legal permission, or World Enabler vetting before a user can be recruited for that task.

## Skill Tiers

### Entry-Level Skills

These are safe for a new user to add immediately because they describe attitude, availability, and willingness rather than specialized authority.

- Enthusiast
- Team spirit
- Willing hands
- Learner
- Friendly welcome
- Setup helper
- Cleanup helper
- Runner

### Earned Traits

These unlock through verified action history, World Enabler ratings, and repeated follow-through.

- Reliability
- Leadership
- Calm under pressure
- Event captain
- Community connector
- Mentor readiness

Suggested rule: require a mix of verified attendance, World Enabler confirmation, and positive post-event feedback. Do not let repeated low-value attendance alone unlock high-trust traits.

### Course-Verified Skills

These require approved training, partner-issued credentials, or Handprint-reviewed coursework.

- First aid
- Lifeguard
- Food safety
- Grant writing
- Proposal writing
- Youth mentoring
- Crisis response

### Legally or Organization-Vetted Skills

These require legal eligibility and World Enabler review before recruitment.

- Driving
- Passenger transport
- Working with minors
- Medical support
- Financial stewardship
- Equipment operation

Driving should not unlock from user self-report alone. It should require license validation, role-specific checks, and World Enabler approval before a user can be recruited to transport people.

## World Enabler Incentive

Handprint should reward World Enablers for welcoming new users into entry-level roles. A healthy pilot should not optimize only for experienced volunteers. World Enablers can earn a "New Hand Builder" accolade when they create beginner-friendly roles, confirm first-time participants, and give useful feedback that helps those users earn their first traits.

## Product Rule

Profile Skills are not one flat list. V1 should treat them as:

- Offered now: safe entry-level skills the user can volunteer for today.
- Earning: traits currently in progress through verified participation.
- Credentialed: skills backed by course, document, or partner verification.
- Vetted: high-responsibility skills approved by a World Enabler or reviewer for specific contexts.
